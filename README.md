## Começando

### Clone o repositório

```shell
git clone https://github.com/Johnson49/reformulado-o-site-da-Gol-Software.git
cd reformulado-o-site-da-Gol-Software
```

### Instale as dependências

```javascript
npm install
```

### Inicie o servidor

```javascript
npm run dev
```
